# 🚀 Smart Study Planner Pro

A modern and visually appealing productivity web app designed to help students manage their study tasks efficiently.

---

## ✨ Features

- ✅ Add and delete study tasks  
- 📅 Set deadlines for tasks  
- 🧠 Categorize tasks (Physics, Chemistry, Maths, etc.)  
- 📊 Progress tracking with progress bar  
- 🔥 Streak system for motivation  
- 📈 Doughnut chart for analytics  
- 💾 Data stored using localStorage  

---

## 🛠️ Tech Stack

- HTML  
- Tailwind CSS  
- JavaScript  
- Chart.js
